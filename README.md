# tinybox Scripts

Useful scripts to setup and manage the tinybox

```
./
├── burn.sh               - Run the GPU stress tests
├── create_raid.sh        - Creates RAID0 disk
├── fan_control.sh        - Sets fan speed
├── get_aers.sh           - Get device ID from kernel messages
├── gpu_reset.sh          - Reset GPU
├── install_nvidia.sh     - Installs NVIDIA drivers
├── limit_power.sh        - Limits GPU power consumption
├── logo                  - Generates logo
│   ├── build.sh          -
│   ├── logo.png          - 
│   └── logo.svg          - 
├── new_user.sh           - Adds new user with the SSH public key
├── screen.py             - Defines different messages for the LCD screen
├── temps.sh              - Prints temperatures
├── tst_disk.sh           - Runs disk tests
├── tst_gpu.sh            - Runs GPU tests
└── tst_raid.sh           - Runs RAID0 tests
```
